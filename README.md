# 🍰 Doce Encanto

### Loja virtual de doces com React

> “Mais do que vender doces, é sobre criar experiências que dão água na boca.” 😋

---

## ✨ Sobre o projeto

O **Doce Encanto** é uma aplicação web desenvolvida com React que simula uma loja virtual de doces, permitindo que os usuários visualizem produtos e realizem pedidos de forma simples e rápida via WhatsApp.

O projeto foi criado com foco em **boas práticas de desenvolvimento front-end**, componentização e experiência do usuário.

---

## 🎯 Objetivo

Este projeto tem como objetivo demonstrar:

* Estruturação de uma aplicação React
* Criação de componentes reutilizáveis
* Organização de código escalável
* Integração com serviços externos (WhatsApp)

---

## 🚀 Funcionalidades

* 🍬 Listagem de produtos
* 🧩 Componentes reutilizáveis (Header e ProductCard)
* 📱 Botão de compra com redirecionamento para WhatsApp
* 💅 Interface simples, moderna e responsiva (base)

---

## 🧱 Tecnologias utilizadas

* ⚛️ React
* ⚡ Vite
* 🎨 CSS3
* 🧠 JavaScript (ES6+)

## ▶️ Como executar o projeto

```bash
# Clone o repositório
git clone https://github.com/seu-usuario/doceria-react.git

# Acesse a pasta
cd doceria-react

# Instale as dependências
npm install

# Execute o projeto
npm run dev
```

Depois, acesse no navegador:

```
http://localhost:5173
```
